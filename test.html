HELP

You can add field to your bnsList request with select=...
You can add some filters too with filter=...

Select syntax

example: select=fieldName,fiedName2,etc...

Filter syntax

syntax construction: filter=fieldName operator value
For multiple conditions add ' and ' then next condition.operator list:
  • ' ne null ' -> is not null
  • ' ne ' -> different from
  • ' eq null ' -> is null
  • ' eq ' -> is equal to
  • ' gt ' -> is greater than
  • ' ge ' -> is greater than or equal
  • ' lt ' -> is less than
  • ' le ' -> is less than or equal
  • ' indexof('fieldName','valueSearch') ->fieldName is included in valueSearch
